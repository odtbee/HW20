Department of Labor Responsive Redesign

A Case Study

The current Department of Labor website is full of important information for workers in the United States.

However, it lacks organization and can be confusing to navigate.

In this responsive redesign, I attempted to alleviate some of these major concerns.

Let's Go!

In order to address some of the outstanding issues with the DOL website, a responsive redesign of the site’s UI was the solution.

RESPONSIBILITIES UI Designer User Testing UI Styling

The Problem:

The current DOL website is an important resource, but can be confusing and overwhelming with the amount of information present.

The Solution:

A responsive redesign of the user interface of the website that addresses visitors’ concerns.

The Process

With a limited timeline, it was important to follow a schedule and focus on the essential processes.

Annoted webpage with analysis

To make an informed redesign, we gathered data through UI analysis against usability heuristics and user tests on the current DOL website.

Ideation and Design

Site navigation and the primary pages of the website were prototyped using Adobe XD. This part of the redesign focused on making the navigation more organized, readable, and intuitive.
